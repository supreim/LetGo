# Let Go

Let Go is a local-first web app for practicing emotional letting go.

It is designed to help people notice a feeling or body sensation, stop resisting it, stay present with it, and reflect on what shifted. The app turns that inner process into something you can return to consistently, track over time, and learn from.

## What "letting go" means here

In this app, letting go does not mean forcing yourself to move on, suppressing emotions, or pretending you are fine.

It means:

1. Bringing a feeling or sensation into awareness
2. Relaxing into it instead of resisting it
3. Letting it be there long enough to be fully felt
4. Noticing whether relief, softness, clarity, or emotional movement appears

The practice is inspired by the idea that emotions often stay stuck when they are resisted, judged, or mentally fought. When they are allowed, they can move through more naturally.

## The process inside the app

Each session helps the user move through a simple sequence:

1. Choose a doorway for bringing up the feeling
2. Start a timed letting-go session
3. Stay with the sensation and allow it
4. End the session and reflect on how it went
5. Save the session so it becomes part of your history, analytics, and calendar

The available doorways are:

- `Shadow work`
- `Picture the worst case`
- `Return to a memory`
- `Do nothing`

Each doorway includes space to capture itemized notes so the user can revisit what brought the sensation up later.

## Who this is for

This app is for people who want a more intentional way to work with emotions, triggers, and recurring inner patterns.

It may be especially useful for people who:

- feel stuck in repeating emotional loops
- want to practice allowing feelings instead of suppressing them
- are interested in self-inquiry, shadow work, and emotional awareness
- want structure around a letting-go practice
- want to track growth over time in a simple, personal way

It is not a replacement for therapy, crisis support, or medical care. It is a self-reflection tool for personal practice.

## What the app helps you track

The app combines real-time practice with longer-term reflection.

### Letting-go sessions

Users can log:

- what they were feeling
- which doorway they used
- how long they spent letting go
- how much they felt they allowed the sensation
- whether they felt relief at the end
- notes about the session

### Separate trackers

Outside of individual sessions, users can keep track of:

- issues they want to let go of
- triggers that keep activating them
- gains they have made through letting go
- their current consciousness level

### Calendar and analytics

The app includes:

- a calendar that shows when letting-go sessions happened
- filtering by whether relief was felt
- analytics for time spent, frequency, allowing, and relief trends
- a graph with toggleable metrics

### Daily reflection journal

The journal gives users a structured daily check-in with prompts such as:

- what feeling came up
- whether they allowed the sensation
- what happened versus what they wanted to happen
- what patterns they recognized
- how they started the day and how they felt by the end

Journal entries are linked to specific days, and the calendar day filter also applies to them.

## What someone may gain from using it

Used consistently, the app can help people:

- build awareness of recurring emotional patterns
- notice what triggers them most often
- practice staying with feelings instead of escaping them
- see whether relief is becoming more common over time
- recognize progress that might otherwise be missed
- develop a calmer, more honest relationship with inner experience

The goal is not perfection. The goal is to become more able to feel, allow, observe, and release.

## Local-first by design

This app currently stores data in the browser using local storage.

That means:

- sessions stay on the device unless exported
- journal entries stay on the device unless exported
- trackers and theme settings stay on the device unless exported
- users can export and import their data as a backup

## Current features

- guided start-session flow
- timed letting-go sessions
- edit and delete for saved sessions
- itemized doorway notes
- separate issue, trigger, gain, and consciousness tracking
- daily reflection journal
- analytics dashboard and graph
- calendar-based filtering
- session history
- import and export
- light mode and dark mode

## Running the app locally

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

Other useful commands:

```bash
npm run lint
npm run build
npm run start
```

## Vision

Let Go is meant to feel like a calm personal practice space, not just a tracker.

The long-term idea is to help people build a gentle but honest relationship with emotional release: noticing what they are carrying, allowing it instead of fighting it, and seeing how that changes over time.
